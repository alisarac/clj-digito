# clj-digito

A clojure library for calling Digital Ocean api.

## Usage

### Requirements:

- token environment variable should be set for authorization, .lein-env file can be used.


### Provided functions are

- api-get
- api-post
- api-delete

Example usage can be seen in core.clj file.
